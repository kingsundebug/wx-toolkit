# 加水印（相册选图 · 文字平铺）设计说明

**日期：** 2026-05-27  
**状态：** 已确认  
**范围：** A 方案 — 独立工具页，与去水印无联动

## 目标

用户从相册选择一张图片，输入水印文字，在 Canvas 上平铺半透明文字后预览，并保存到系统相册。

## 不在 MVP

- Logo / 图片水印
- 与去水印解析结果联动
- 单点水印、拖拽、自定义字体、批量处理
- 视频加水印

## 架构

- **页面：** `pages/add-wm/add-wm.vue` — 选图、参数、预览、保存
- **绘制：** `common/watermark-draw.js` — `getImageInfo` → 等比缩放到最长边 ≤2048 → `drawImage` + 旋转平铺 `fillText` → `canvasToTempFilePath`
- **注册：** `common/tools.js` 中 `add-wm` 设为 `ready`，`pages.json` 注册路由

## 用户流程

1. 进入「加水印」页 → 选择图片（`uni.chooseImage`，单张）
2. 输入水印文字（默认「仅供预览」）、调节透明度（约 10%–50%）
3. 生成预览（Canvas）
4. 保存到相册（`scope.writePhotosAlbum`，与二维码页相同鉴权模式）

## 绘制参数（默认）

| 参数 | 默认 |
|------|------|
| 透明度 | 0.25 |
| 旋转角 | -30° |
| 字号 | 按画布短边比例计算 |
| 平铺间距 | gapX 160 / gapY 100（px） |

## 错误处理

- 未选图 / 空文字 → Toast 提示
- `getImageInfo` / 绘制 / 导出失败 → Toast，不崩溃
- 相册未授权 → 弹窗引导 `openSetting`
